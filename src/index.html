Testing HTML5 web workers

This small app computes the sum of the natural numbers from 1 to 3333333333. The result is 5555555554511501000 (it is a nice number, isn't it?). Normally this operation lasts about 6-7 seconds. As you can see such a computation consideralibly blocks the responsiveness of the page. One of the solution of avoiding this blocking effect is using web workers.

Sorry, but your browser doesn't support the HTML5 webworkers API. Please try the app on an other browser (Internet Explorer 10+, Firefox 3.5+, Opera 10.60+, Chrome 3+, Safari 4+)
